Plugin to show license agreements on your website.

== Description ==

Do you have a license agreement that you visitor has to accept before he can view your content? Or do you need to inform your visitor about the usage of cookies on your site?

Then this plugin is perfect for you. We allow you to show you visitors a disclaimer in an elegant popup/overlayer. Your visitor will have to accept this agreement before he can continue to your website.

A custom admin menu lets you modify the text and some extra settings.

An example is available on http://mindloop.be/disclaimer/

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder `disclaimer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You'll find a new menu called `disclaimer` in your Wordpress admin, go here to configure the plugin
